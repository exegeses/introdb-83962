# Crear bases de datos en SQL

> Para crear una base de datos en SQL utilizamos el comando **CREATE**

> Sintaxis:  

    CREATE DATABASE nombreBase;  

> Para este curso vamos a referirnos todo el tiempo a una base de datos llamada "introdb"

> Ejemplo práctico: 

    CREATE DATABASE introdb;  

## Eliminar una base de datos

> Para elimnar una base de datos en SQL utilizamos el comando **DROP**

> Sintaxis:

    DROP DATABASE nombreBase;  

> Ejemplo práctico: 

    DROP DATABASE introdb;
